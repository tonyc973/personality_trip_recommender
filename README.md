# Personality Trip Recommender

This application recommends travel destinations based on the user's personality traits. Using behavioral analysis, and leveraging web searches, it suggests trips that are aligned with users' preferences and generates a markdown report.

## Setup

1. Clone the repository.
2. Install dependencies using Poetry:
   ```bash
   poetry install
   ```
3. Create a `.env` file from `.env.example` and fill in the required API keys.

## Running the Application

Execute the main.py with the desired user input:

```bash
python src/personality_trip_recommender/main.py
```

## Example Output

- A markdown report listing travel recommendations tailored to your personality traits.
